# AGENTS.md

## Objetivo

Este proyecto implementa un sistema multiagente para diseñar arquitecturas de software.

## Principios

- Priorizar código limpio.
- Seguir principios SOLID.
- No duplicar lógica.
- Documentar las decisiones importantes.
- Nunca eliminar archivos sin autorización.
- Siempre explicar cambios complejos.

## Tecnologías

Frontend:
- React
- TypeScript
- Tailwind

Backend:
- NestJS
- mariaDB

## Estilo

- Código en inglés.
- Documentación en español.
- Comentarios mínimos.
